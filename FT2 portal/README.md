# FT2_portal

An Aardwolf MUD plugin designed to assist with the Fairy Tales II goal through automated room instructions, interactive hyperlinks, and a dedicated shopping guide miniwindow.

## Features
- **GMCP Integration:** Triggers instructions and clickable links automatically based on your room ID (Tasks 9-21).
- **Interactive Miniwindow:** Quickly navigate to shops for required items like dirks, hats, and chocolate.
- **Goal Shortcuts:** Clickable links for complex sequences like the wolf's teeth, goat meat, and the phoenix feather.
- **Auto-Update:** Keep the plugin current using the built-in update command.

## Commands
- `ft2 guide on`  : Displays the FT2 Guide window.
- `ft2 guide off` : Hides the FT2 Guide window.
- `ft2 update`    : Checks GitHub for the latest version and installs.
